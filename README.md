# Power BI Sales Project Documentation

## 1. Project Title
**Sales Performance Analysis using Power BI**

## 2. Project Overview

This project focuses on analyzing sales data using **Microsoft Excel** And **Microsoft Power BI** to derive meaningful business insights. The objective is to transform raw sales data into interactive and visually appealing dashboards that help stakeholders understand overall sales performance, regional trends, and product/category-level insights.

The project is built using a provided dataset and consists of **three main dashboard pages**:

1. Sales Overview Dashboard
2. Regional Analysis Dashboard
3. Product & Category Insights Dashboard

## 3. Objectives of the Project

* To analyze overall sales performance using key performance indicators (KPIs)
* To identify high-performing and low-performing regions
* To understand product-wise and category-wise sales trends
* To support data-driven decision-making through interactive dashboards

## 4. Tools & Technologies Used

* **Microsoft Excel** – Data cleaning and transformation
* **Microsoft Power BI** – Data modeling, visualization, and dashboard creation
* **DAX (Data Analysis Expressions)** – Calculated measures and KPIs
* **Excel/CSV Dataset** – Source data provided for analysis

## 5. Dataset Description

The dataset includes sales-related information such as:

* Order ID
* Order Date
* Region
* State / City
* Product Name
* Product Category
* Sales Amount
* Quantity Sold
* Profit 

The dataset was cleaned and transformed to ensure data accuracy and consistency before visualization.

## 6. Data Preparation & Modeling

* Removed duplicate and null values using Power Query
* Converted data types (date, numeric, text)
* Created relationships between tables (if multiple tables were used)
* Created calculated columns and measures using DAX

### Key DAX Measures Used:

* Total Sales
* Total Quantity
* Total Profit
* Average Sales

## 7. Dashboard Pages Description

### 7.1 Sales Overview Dashboard

**Purpose:**
Provides a high-level summary of overall sales performance.

**Key Visuals Used:**

* KPI Cards (Total Sales, Total Quantity, Total Profit)
* Line Chart for sales trend over time
* Bar/Column Chart for sales by category
* Slicers for Date, Region, and Category

**Insights Gained:**

* Overall business performance
* Sales trend patterns
* Comparison of sales across categories

---

### 7.2 Regional Analysis Dashboard

**Purpose:**
Analyzes sales performance across different regions.

**Key Visuals Used:**

* Map Visualization (Sales by Region/State)
* Bar Chart for regional sales comparison
* Table showing region-wise sales and profit
* Slicers for region and date

**Insights Gained:**

* Best and worst performing regions
* Regional contribution to total sales
* Geographic sales distribution

---

### 7.3 Product & Category Insights Dashboard

**Purpose:**
Provides detailed insights into product-level and category-level performance.

**Key Visuals Used:**

* Bar Chart for top-selling products
* Donut/Pie Chart for category-wise sales share
* Table showing product-wise sales and quantity
* Filters for category and product

**Insights Gained:**

* Top-performing products
* Category contribution to revenue
* Products requiring improvement or promotion

## 8. Interactivity Features

* Slicers for dynamic filtering
* Cross-filtering between visuals
* Drill-down and drill-through 
* Tooltips for better data understanding
